# 🌾 SmartAgriAI – Intelligent Agriculture Assistant

> 🚀 A unified AI-powered platform integrating Machine Learning & Deep Learning models to assist farmers and agri-enthusiasts in making smarter decisions.

---

## 📌 Overview

SmartAgriAI is a centralized web platform that combines multiple AI models into one ecosystem. It provides intelligent predictions related to agriculture such as crop recommendation, disease detection, and yield prediction.

🔗 All models are deployed individually and accessible through a single home interface.

---

## 🎯 Key Features

- 🌱 **Crop Recommendation System** (ML)
- 🦠 **Plant Disease Classification** (Deep Learning)
- 🌾 **Yield Prediction** (ML)
- 📊 **Additional ML Model** (based on agricultural data)
- 🌐 **Centralized Home Page** for navigation
- 🚀 Fully deployed applications (accessible via links)

---

## 🧠 Models Used

### 🔹 Machine Learning Models
- Random Forest
- Decision Tree
- Logistic Regression
- Support Vector Machine (SVM)

### 🔹 Deep Learning Model
- Convolutional Neural Network (CNN) for image classification

## 🚀 How to Run Locally

**1️⃣ Clone the repository**
```bash
git clone https://github.com/rajsirohi23/SmartAgriAI.git
cd SmartAgriAI
```

**2️⃣ Create virtual environment**
```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

**3️⃣ Install dependencies**
```bash
pip install -r requirements.txt
```

**4️⃣ Run the application**
```bash
python app.py
```
